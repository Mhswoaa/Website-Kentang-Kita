# Website-Kentang-Kita
Website resmi Kentang Kita. Kentang Kita merupakan UMKM kuliner yang menjual kentang goreng, kulit, usus, dan jamur crispy. Dibuat sebagai landing page modern &amp; responsif menggunakan HTML, CSS, dan JavaScript. Fitur meliputi menu interaktif, blog, testimoni, FAQ, serta integrasi WhatsApp Order &amp; platform delivery (GoFood &amp; ShopeeFood).
